## 🚀 Highlighted Projects

🩺 Clinical Text Classification Mini-Pipeline (Healthcare NLP)
End-to-end NLP pipeline for multi-class classification of medical abstracts (PubMed 20k RCT dataset).

**What this project demonstrates:**
- Data preprocessing and exploratory analysis of clinical text
- TF-IDF + Logistic Regression baseline (~83% accuracy)
- DistilBERT fine-tuning (~87% accuracy)
- Per-class evaluation using precision, recall, F1-score
- Confusion matrix analysis of label overlap (e.g., OBJECTIVE vs BACKGROUND)
- Reproducible, research-oriented experimentation workflow

This project reflects my interest in clinical NLP and interpretable machine learning for healthcare.

👁️ Real-Time Computer Vision & Gesture Interaction
Webcam-based computer vision system enabling touchless human-computer interaction.

**Key aspects:**
- Grayscale conversion and RGB channel analysis
- HSV & YCbCr colour space transformations
- Adaptive thresholding and segmentation
- Face detection with region-based pixel transformations
- Gesture-controlled drawing using centroid tracking
- Real-time performance optimization

This project demonstrates applied computer vision, image processing, and interactive system design.
